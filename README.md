# 💸 SpendSmart

A personal expense tracker to manage and visualize your spending habits.

## 🛠️ Tech Stack
- HTML, CSS, JavaScript

## ✨ Features
- Add/delete expenses
- Category-wise tracking
- Visual charts

## 🏃 How to Run Locally
1. Clone the repo: `git clone https://github.com/PalakA321/SpendSmart`
2. Open `index.html` in browser
